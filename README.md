# Sterling & Associates Law Firm Portfolio

A professional law firm portfolio website featuring a modern, elegant design with a dark theme and gold accents.

## Features

- Responsive design
- Practice areas showcase
- Attorney profiles
- Case results
- Contact form
- Smooth animations and transitions
